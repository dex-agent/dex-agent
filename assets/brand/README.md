# Dex Agent Brand Assets

Versionable logo assets for GitHub, docs, and package pages.

- `dex-agent-logo.svg`: horizontal wordmark for README headers and social surfaces.
- `dex-agent-mark.svg`: compact mark for icons, avatars, and square placements.
- `dex-agent-logo.png`: PNG export of the horizontal wordmark.
- `dex-agent-mark.png`: PNG export of the compact mark.

The SVG files are the source assets. PNG files are reviewed exports for GitHub surfaces that do not accept SVG. Keep generated local drafts and Canva downloads out of Git unless they are intentionally reviewed brand assets.
